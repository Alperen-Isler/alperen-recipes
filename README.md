# Alperen's recipes

This is my first project from **The Odin Project** curriculum.  
The goal of this project is to build a simple recipe website using only HTML.

## Project Description

The website consists of a main index page that links to several recipe pages.  
Each recipe page includes:
- a heading
- an image of the dish
- a list of ingredients
- step-by-step cooking instructions
